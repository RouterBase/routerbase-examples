# Security

## Reporting Issues

Please report security issues privately through the RouterBase website:

https://routerbase.com

Do not open public issues for secrets, authentication problems, or vulnerabilities.

## Secrets

Examples use the `ROUTERBASE_API_KEY` environment variable. Do not commit real API keys, npm tokens, PyPI tokens, or GitHub tokens.

## Supported Versions

This repository is currently in early `0.x` development. Security fixes are applied to the latest version on `main`.
